1. app 프로젝트 생성
```shell
git init
git add ./* 
git commit -m "init"
git remote 등록 
git remote push
```
2. File -> new -> new module -> android library로 프로젝트 생성
3. library 폴더에서
```shell
git init
git add ./* 
git commit -m "init"
git remote 등록 
git remote push
```
4. git submodule 등록
```shell
git submodule add {GIT SUBMODULE REMOTE PATH} {library 프로젝트 path}
```

하면 끝!
